
# 1022 D进制的A+B

## Statement

!!! info "Metadata"
    - 作者: CHEN, Yue

    - 代码长度限制: 16 KB
    - 时间限制: 200 ms

输入两个非负 10 进制整数 $A$ 和 $B$ ($\le 2^{30} -1$)，输出 $A+B$ 的 $D$ ($1 < D \le 10$)进制数。

**输入格式**

输入在一行中依次给出 3 个整数 $A$、$B$ 和 $D$。

**输出格式**

输出 $A+B$ 的 $D$ 进制数。

**输入样例**
```plaintext
123 456 8

```

**输出样例**
```plaintext
1103

```
