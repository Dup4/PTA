
# 1017 A除以B

## Statement

!!! info "Metadata"
    - 作者: CHEN, Yue
    - 单位: 浙江大学
    - 代码长度限制: 16 KB
    - 时间限制: 100 ms
    - 内存限制: 64 MB

本题要求计算 $A/B$，其中 $A$ 是不超过 1000 位的正整数，$B$ 是 1 位正整数。你需要输出商数 $Q$ 和余数 $R$，使得 $A = B \times Q + R$ 成立。

**输入格式**

输入在一行中依次给出 $A$ 和 $B$，中间以 1 空格分隔。

**输出格式**

在一行中依次输出 $Q$ 和 $R$，中间以 1 空格分隔。

**输入样例**
```plaintext
123456789050987654321 7
```

**输出样例**
```plaintext
17636684150141093474 3
```
