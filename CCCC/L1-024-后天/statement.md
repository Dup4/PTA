
# L1-024 后天

## Statement

!!! info "Metadata"
    - 作者: 陈越

    - 代码长度限制: 16 KB
    - 时间限制: 400 ms

如果今天是星期三，后天就是星期五；如果今天是星期六，后天就是星期一。我们用数字1到7对应星期一到星期日。给定某一天，请你输出那天的“后天”是星期几。

**输入格式**

输入第一行给出一个正整数`D`（1 $\le$ `D` $\le$ 7），代表星期里的某一天。

**输出格式**

在一行中输出`D`天的后天是星期几。

**输入样例**
```plaintext
3
```

**输出样例**
```plaintext
5
```
