
# L2-008 最长对称子串

## Statement

!!! info "Metadata"
    - 作者: 陈越
    - 单位: 浙江大学
    - 代码长度限制: 16 KB
    - 时间限制: 200 ms
    - 内存限制: 64 MB

对给定的字符串，本题要求你输出最长对称子串的长度。例如，给定`Is PAT&TAP symmetric?`，最长对称子串为`s PAT&TAP s`，于是你应该输出11。

**输入格式**

输入在一行中给出长度不超过1000的非空字符串。

**输出格式**

在一行中输出最长对称子串的长度。

**输入样例**
```plaintext
Is PAT&TAP symmetric?
```

**输出样例**
```plaintext
11
```
