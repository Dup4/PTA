
# L1-013 计算阶乘和

## Statement

!!! info "Metadata"
    作者: 陈越
    单位: 浙江大学
    代码长度限制: 16 KB
    时间限制: 400 ms
    内存限制: 64 MB

对于给定的正整数$$N$$，需要你计算 $$S = 1! + 2! + 3! + ... + N!$$。

### 输入格式：

输入在一行中给出一个不超过10的正整数$$N$$。

### 输出格式：

在一行中输出$$S$$的值。

### 输入样例：
```plaintext
3
```

### 输出样例：
```plaintext
9
```
