
# L1-039 古风排版

## Statement

!!! info "Metadata"
    - 作者: 陈越
    - 单位: 浙江大学
    - 代码长度限制: 16 KB
    - 时间限制: 400 ms
    - 内存限制: 64 MB

中国的古人写文字，是从右向左竖向排版的。本题就请你编写程序，把一段文字按古风排版。

**输入格式**

输入在第一行给出一个正整数$N$（$<100$），是每一列的字符数。第二行给出一个长度不超过1000的非空字符串，以回车结束。

**输出格式**

按古风格式排版给定的字符串，每列$N$个字符（除了最后一列可能不足$N$个）。

**输入样例**
```plaintext
4
This is a test case
```

**输出样例**
```plaintext
asa T
st ih
e tsi
 ce s
```
