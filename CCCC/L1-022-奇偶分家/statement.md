
# L1-022 奇偶分家

## Statement

!!! info "Metadata"
    - 作者: 陈越
    - 单位: 浙江大学
    - 代码长度限制: 16 KB
    - 时间限制: 400 ms
    - 内存限制: 64 MB

给定`N`个正整数，请统计奇数和偶数各有多少个？

**输入格式**

输入第一行给出一个正整`N`（$\le 1000$）；第2行给出`N`个非负整数，以空格分隔。

**输出格式**

在一行中先后输出奇数的个数、偶数的个数。中间以1个空格分隔。

**输入样例**
```plaintext
9
88 74 101 26 15 0 34 22 77
```

**输出样例**
```plaintext
3 6
```
